Android Open Source Apps

  • Wed, May 6, 2015
  • 2-minute read

In order to improve my skills on android development I always search for sample code and valueable apps from which learning new things. Here is my list of android apps I found with source code available. This post has been inspired from this article about great android apps that are also open source.

wall:splash

wall:splash is a free open source android client for the awesome Unsplash service (Free (do whatever you want) high-resolution photos) developed by Mike Penz.
